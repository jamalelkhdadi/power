Syntax

data contains key => value

key and value separated by colon : // key : value

key should be wrapped in double quotes // "key" : "value"

separate data by comma (,) { "key" : "value" , "key" : "value"}

can be written in one line or multiple lines depend on data

square brakets contains arrays [ ]

curly braces containes objects { }
